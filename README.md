
## Задача

Нужно дописать файлы GitHub Actions так, чтобы проект автоматически проверялся и создавал artifact.

## Что нужно сделать

1. Дополнить файл `.github/workflows/build.yml`.
2. Добавить в нем запуск тестов.
3. Добавить job для события открытия pull request.
4. Дополнить файл `.github/workflows/report.yml`.
5. Настроить сохранение artifact.
6. Дополнить файл `.github/workflows/manual.yml`.
7. Запустить workflow вручную и проверить input.

## Подсказка

Код проекта уже готов. Менять Kotlin-файлы не нужно.

Главная папка для работы:

`student-template/.github/workflows/`
